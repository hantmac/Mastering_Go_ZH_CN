# **代码规范约定**

下面是关于本书的代码规范约定。

文本框中的内容：代码，数据库表名，文件夹名，文件名，扩展名，路径名，URL，用户输入等。

下面是代码块示例：

```go
package main
import (
    "fmt"
)
func main() {
    fmt.Println("This is a sample Go program!")
}
```

代码块中强调的代码，会用下划线或者加粗的形式给出：

>import (
>"fmt"
>)
>func main() {
>
>**fmt.Println("This is a sample Go program!")**

>}
>
>

命令行输入输出的格式如下所示：

> $ date
>
> Sat Oct 21 20:09:20 EEST 2017
>
> $ go version
>
> go version go1.9.1 darwin/amd64
>
>

**粗体**：意味着这是重要的内容，或者是从屏幕中输出的内容。
